**쿼리스트링이란**
URL 뒤에 입력 데이터 제공하는 가장 단순한 데이터 전달 방법.
데이터를 요청하는 방식 중 GET 방식으로 데이터를 요청할 때 사용한다.
URL주소뒤에 물음표를 붙이고 key1=value1&key2=value2...방식으로 데이터를 요청한다.
