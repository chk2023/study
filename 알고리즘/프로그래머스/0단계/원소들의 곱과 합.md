```
public class 원소들의_곱과합 {  
    public int solution(int[] num_list) {  
        int sum = 0;  
        int product = 1;  
  
        for(int num : num_list){  
            sum += num;  
            product *= num;  
        }  
  
        if(product < sum*sum){  
            return 1;  
        } else {  
            return 0;  
        }  
    }  
}
```
for-each 루프num_list의 모든 요소를 int num이라는 변수에 하나씩 반복해서 대입.
