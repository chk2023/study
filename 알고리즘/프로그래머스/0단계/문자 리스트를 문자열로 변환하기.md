```
class Solution {
    public String solution(String[] arr) {
        return String.join("", arr);
    }
}
```
해결 1.
문자열 배열을 입력으로 받아, 그 배열의 모든 원소들을 이어붙여 하나의 문자열로 만든다.
String.join() 메서드는 문자열 배열의 모든 요소를 하나의 문자열로 연결. 


```
class Solution {
    public String solution(String[] arr) {
        String answer = "";

        for(String a : arr) {
            answer += a;
        }

        return answer;
    }
}
```

```
class Solution {
    public String solution(String[] arr) {

        }
        return sb.toString();
    }
```
