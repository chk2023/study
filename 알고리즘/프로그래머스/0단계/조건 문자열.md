```
class Solution {
    public int solution(String ineq, String eq, int n, int m) {
        if ("<".equals(ineq)) {
            if ("=".equals(eq)) {
                return n <= m ? 1 : 0;
            } else if ("!".equals(eq)) {
                return n < m ? 1 : 0;
            }
        } else if (">".equals(ineq)) {
            if ("=".equals(eq)) {
                return n >= m ? 1 : 0;
            } else if ("!".equals(eq)) {
                return n > m ? 1 : 0;
            }
        }
        return -1;
    }
}
```
조건이 두 개이므로 if문 안의 if문을 사용.
예) 조건1 ">"와 같으면 그 다음 조건2 "="와 같은지 확인 후 결과 값 반환.
